# Registration Form Project

## Description
A simple registration form built using HTML, CSS, and JavaScript. Includes client-side validation for email, phone number, and password confirmation. Demonstrates frontend development skills and user interaction.

## Features
- First Name, Last Name, Email, Phone, Password, and Confirm Password fields
- Gender selection using radio buttons
- Validation for required fields, email format, phone number, and matching passwords
- Alerts for success and error messages
- Responsive and visually appealing UI using CSS flexbox

## Skills Used
- HTML for form structure
- CSS for styling and layout
- JavaScript for form validation and user feedback

## How to Run
1.Using vs code
2. Save `index.html`, `reg.css`, and `reg.js` in the same folder.
3. Open `index.html` in a web browser.
4. Fill the registration form and submit.
